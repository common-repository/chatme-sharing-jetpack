== Description ==

Extends the Jetpack plugin and allows you to add a ChatMe sharing button to the list of sharing services available under Settings > Sharing in your dashboard.

Important: for this plugin to work, you must activate Jetpack first, and activate the Sharing module.

== Installation ==

= From your WordPress dashboard =

1. Visit 'Plugins > Add New'
2. Search for 'ChatMe Sharing JetPack'
3. Activate ChatMe Sharing JetPack from your Plugins page.
4. Click Options from your Plugin page

= From WordPress.org =

1. Download ChatMe Sharing JetPack.
2. Upload the 'chatme-sharing-jetpack' directory to your '/wp-content/plugins/' directory, using your favorite method (ftp, sftp, scp, etc...)
3. Activate ChatMe Sharing JetPack from your Plugins page.
4. Click Options from your Plugin page
